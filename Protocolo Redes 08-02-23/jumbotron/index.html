Protocolo de Rede

Protocolos de Rede são um conjunto de normas que permitem que qualquer máquina conectada à internet possa se comunicar com outra também já conectada na rede, esses protocolos funcionam como uma linguagem universal que permite que computadores de fabricantes e sistemas operacionais diferentes se comuniquem sem problemas, cada tipo de protocolo possui características e funções específicas, que permitem que os computadores se comuniquem de forma rápida, fácil e segura, além disso, esses protocolos são divididos em três elementos-chave, que são sintaxe, semântica e sincronização.

ARP

O protocolo ARP, permite encontrar o endereço físico a partir do endereço IP da máquina alvo. Para tal, o protocolo usa um mecanismo de difusão (broadcast) na rede local, enviando uma solicitação a todas as máquinas da rede, sendo que a máquina alvo responde indicando o par endereço IP/endereço físico.

HTTP

O HTTP (Hypertext Transfer Protocol) descreve um protocolo sem estado com o qual os dados podem ser transmitidos em uma rede IP. A aplicação mais importante é a transferência de páginas da Internet e dados entre um servidor web e um navegador.
